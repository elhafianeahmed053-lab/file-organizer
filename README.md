# File Organizer (Python)

This Python script automatically organizes files in a folder by extension.

## Features
- Creates folders for each file type
- Moves files automatically
- Handles files with no extension

## Example

Before:

file1.txt  
image.png  
script.py  

After:

txt/  
png/  
py/  

## Run the program

```bash
python3 organize_files.py
